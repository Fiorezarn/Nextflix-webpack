## Cara Menjalankan Aplikasi
- Clone repository ini ke komputer Anda.
- Buka terminal dan arahkan ke direktori proyek.
- Jalankan perintah
```sh
npm install
```
untuk menginstal semua dependensi.
- 
```sh
npm run build
```
untuk mengkompilasi kode JavaScript.
-
```sh
npm start dev
```
untuk memulai server.
